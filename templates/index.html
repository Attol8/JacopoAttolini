Photo2Painting

Turn landscapes pictures into old masters' paintings

The App is a PyTorch implementation of the notorious CycleGAN paper by Zhu et al. (2017) "Unpaired Image-to-Image Translation using Cycle-Consistent Adversarial Networks".
I have used and re-trained models that can be found on the official GitHub page of the project to create an easy to use web application that transforms landscape pictures into paintings of three old masters:

  • Van Gogh
  • Cezanne
  • Monet
The web application was built with Flask (a micro web framework written in Python) and deployed on a DigitalOcean droplet. The project wants to demonstrate its users how Generative Adversarial Networks works in production and allowed me to dive deep into the theory of GANs.

Unemployment in Italy

A Data Science project

A full-stack data science project that looks to investigate unemployment in Italy. I have collected and scraped data, built models to predict for unemployment (that reach 85% accuracy) at the individual level and deployed a web-application to explore the results (under construction).

Kaggle Kernel

Paintings classifier and Guided Back Propagation

Practising Deep Learning with paintings dataset

The following Kernel looks into developing a paintings classifier based on best-artworks-of-all-time, a Kaggle public dataset with paintings and work of arts (drawings and sketches are included) from 50 of the best artists of all time.

The obvious aim of the classifier is to output the name of the artist that painted the input painting. However, I also believe that an important part of the Kernel can be found in the model interpretation part as we dig deeper into how the model predicted a certain output. The techniques used to evauate and visualize the results of the CNN are Heatmaps and Guided Backpropagation.
